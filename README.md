# Desarrollo Python

Esta carpeta contiene todo el código fuente de Atlas Core.

Todo el desarrollo se realiza inicialmente en el PC utilizando Visual Studio Code.

Posteriormente el código se sincronizará con la Raspberry Pi donde será ejecutado permanentemente.

# Atlas Core

Atlas Core es el núcleo del Proyecto Atlas.

Desde aquí se controlarán todos los componentes del asistente personal:

- Inteligencia Artificial
- Memoria
- Automatización
- API
- Home Assistant
- Arduino
- Voz
- Aplicación Web

Todo el proyecto está desarrollado en Python y pensado para ejecutarse en Docker sobre Raspberry Pi.

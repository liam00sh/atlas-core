# Contexto de IA

Este directorio preparará el contexto que se enviará a los modelos.

## Contexto previsto

- Mensajes recientes.
- Usuario activo.
- Información relevante de memoria.
- Capacidades reales del sistema.
- Personalidad de Daxter.
- Estado de tareas y proyectos.

## Estado

Actualmente solo puede almacenar mensajes temporalmente.

No se utiliza todavía desde `Atlas.process()`.

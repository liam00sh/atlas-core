# Herramientas para la IA

Este directorio contendrá las herramientas que los modelos podrán pedir a Atlas que ejecute.

## Ejemplos futuros

- Consultar recuerdos.
- Ejecutar comandos.
- Obtener datos del sistema.
- Realizar automatizaciones.
- Consultar Home Assistant.

## Seguridad

El modelo no ejecutará herramientas directamente.

Atlas deberá:

1. Validar la solicitud.
2. Comprobar permisos.
3. Pedir confirmación cuando sea necesario.
4. Ejecutar la acción.
5. Devolver el resultado al modelo.

## Estado

No existen herramientas conectadas todavía.
